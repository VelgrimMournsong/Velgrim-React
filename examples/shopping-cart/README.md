# RxJS Shopping Cart Example

## Available Scripts

Run scripts from the repository root with pnpm:

### `pnpm --filter shopping-cart start`

Runs the app in development mode with Vite.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.<br>
You will also see build errors in the console.

### `pnpm --filter shopping-cart build`

Typechecks the example and creates a production bundle.
